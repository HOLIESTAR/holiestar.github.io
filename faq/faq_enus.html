Hi, what can we help you with?

Dear users, thank you for using Message Peeping Tom. If you encounter difficulties while using the app, please check the following common solutions first. We hope you enjoy using our service.

Reminder

If all the methods below are ineffective, please try restarting your phone

1. Can't receive messages?

Confirm that Notification Access Permission is enabled, tap here to enable

For devices running Android 15 and later, the "Enhanced notifications" setting on some phones might interfere with notification delivery. Click here to disable Enhanced notifications
System Settings -> Notifications -> Advanced settings -> "Enhanced notifications"
Depending on the manufacturer, the setting name might be referred to as: "Enhanced notifications" or "Suggest actions and replies for notifications"
The description for this setting is: Get suggested actions, replies, and more.

2. Messaging App Settings

In the settings of messaging apps (WhatsApp, Instagram, Messenger, Telegram, etc.) confirm the following features are enabled:
Enable ✓"Notifications" or "Alerts"
Enable ✓"Show message content" or "Message preview"

3. Why do I always receive "You have new messages"?

Please ensure your messaging apps have notification features properly enabled; when receiving messages, they should display in the system notification panel and show complete message content for normal operation.

4. Maintain stable network connection

MPT can only receive notifications displayed in the notification panel. If many messages come in when you enable network functionality, but your messaging apps only show the last message, we can only capture that last message. Therefore, maintaining a stable network connection allows receiving most messages. Additionally, if your phone is in power saving mode, the network might be disabled during standby, preventing message reception.

5. Battery-related system settings

As mentioned above, for smooth message capture when network is stable, please ensure your phone's power saving settings exclude MPT, WhatsApp, Instagram, Facebook, Telegram, LINE and other messaging apps you need supported.

6. Unable to receive messages after using for a while?

Some phone systems automatically clear background services of other applications periodically for performance and energy saving. MPT may also be cleared. To ensure MPT services operate normally, confirm that your phone's power saving features have excluded MPT, or restart your phone.

7. What apps might affect MPT's operation?

If you have installed power saving, antivirus, performance optimization or similar apps, these apps might stop MPT's background services to speed up system performance. Please set MPT, WhatsApp, Instagram, Facebook, Telegram, and other messaging apps you need supported as whitelisted.

8. Why are group messages categorized as personal messages?

MPT can only receive notifications displayed in the notification panel. You'll notice that messaging apps sometimes don't show which group messages come from when displaying in the notification panel, so MPT cannot identify which group the message comes from. This is a limitation we cannot modify unless messaging apps improve their notification design.

9. Why can't images, videos, and voice messages be supported?

Due to Android's comprehensive security system, MPT cannot access more private messages from messaging software. We deeply regret this limitation.

10. Why doesn't MPT receive messages when the desktop app is open?

When you open a messaging app's desktop version (e.g., WhatsApp, Instagram, Telegram, LINE), it often automatically disables mobile notifications to avoid duplicate alerts.

To ensure MPT works, you must manually change this setting in the desktop application.

Go to the desktop app's Settings → Notifications, and uncheck the option:

"Turn off mobile notifications when using the PC/Desktop version"

Disabling this ensures notifications are still sent to your mobile phone, allowing MPT to receive them.

11. Why are messages incomplete?

MPT can only receive notifications displayed in the notification panel. If the maximum message length provided by messaging apps is shorter than the actual message, this situation occurs. Different messaging apps have different operating mechanisms, and we regret this limitation.
